SDUX OS

A Future Experimental Operating System

About SDUX

SDUX is a future operating system project focused on deep system understanding, performance, and minimalism. It is designed to explore how operating systems work from the lowest level to the graphical interface.

SDUX Desktop Environment

The SDUX Desktop Environment is the graphical core of SDUX OS. Built using Python and Qt, it includes:

  • • Custom Desktop Shell
  • • Integrated Terminal
  • • File Manager
  • • Embedded Web Browser
  • • Linux System Integration

Vision

SDUX aims to become a fully independent operating system environment, built step by step with a focus on engineering precision and system-level knowledge.
